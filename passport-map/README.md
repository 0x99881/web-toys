# 护照签证地图

一个随手练前端做的小项目。

主要就是拿护照签证数据，做一个能看 2D 地图和 3D 地球的小网页。

## 实现了什么

- 可以选择不同护照
- 可以切换 `2D 地图` 和 `3D 地球`
- 不同签证状态会显示不同颜色
- 鼠标放到国家上会显示国家名和签证状态
- 底部可以看当前护照的免签、落地签、电子签、需签证数量
- 可以点底部图例和数字做筛选

## 怎么用

1. 双击项目里的 `start.bat`
2. 等它自动打开浏览器
3. 选护照，然后看地图

如果浏览器没有自动打开：

- 看黑色窗口第一行地址
- 把那个地址复制到浏览器里打开

## 这是个什么项目

这个项目没有后端，没有数据库，也不是什么正式产品。
